12.11. Mixed-Up Code Questions¶

The following code should use a regular expression to find and print the phrase Good morning! in the list greetings, if it is included in the list.

        import re
---
greetings = ["Hello!", "Good Morning!", "good morning!"]
---
for item in greetings:
---
    if re.search('Good morning!', item):
---
    if re('Good Morning!', item): #paired
---
        print(item)
---
        print(greetings) #paired

The following code should use regular expressions to print each item in the list greetings, if it starts with an H or an h.

        import re
---
greetings = ["Hello!", "hello.", "Morning!", "hi"]
---
for item in greetings:
---
    if re.search('^[Hh]', item):
---
    if re('^H,h'), item): #paired
---
        print(item)
