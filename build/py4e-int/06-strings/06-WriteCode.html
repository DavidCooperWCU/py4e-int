Write Code Questions¶

  1. Fix line 2 so that it prints “Hi” instead of “hi”.

  2. Fix the code so that only “meow” is printed.

  3. Write the code to evaluate the length of the string “I like green eggs” and print it. It should print “The length is 17”.

  4. Create a function named count that accepts a string and a letter as arguments, then returns the count of that letter in the string. For example, if the function call was count(“banana”, “a”) it would return 3.

  5. Take the following Python code that stores a string: string = "X-DSPAM-Confidence: 0.8475" Use find and string slicing to extract the portion of the string after the colon character and then use the float function to convert the extracted string into a floating point number called num.

  6. In Robert McCloskey’s book Make Way for Ducklings, the names of the ducklings are Jack, Kack, Lack, Mack, Nack, Ouack, Pack, and Quack. The following loop tries to output these names in order. Of course, that’s not quite right because Ouack and Quack are misspelled. Can you fix it?

  7. Write a function that will return the number of digits in an integer

  8. Write code to print out the statement “Hi my name is Bob and I am 2” using only string methods or string slicing. You must get every part of the new string from the given strings. Name the final string statement.

  9. Write a program that asks a user for their name and from the input prints the first letter of their name in lowercase.

  10. Write a program that gets user input and print a string that states their input in all lowercase and gives the length of their string.
