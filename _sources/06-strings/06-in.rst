The ``in`` operator
-----------------------
The word ``in`` is a boolean operator that takes two strings
and returns ``True`` if the first appears as a substring in the
second:

.. code-block:: python

   >>> 'a' in 'banana'
   True
   >>> 'seed' in 'banana'
   False
